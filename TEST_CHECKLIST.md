## First launch
- [ ] App opens without crash
- [ ] Class selection appears
- [ ] Select Class 6

- [ ] Close and reopen app
- [ ] Selected class is remembered

## Lessons
- [ ] Open a unit

- [ ] Example displays clearly
- [ ] Start quiz
- [ ] Answer all 3 questions
- [ ] Lesson complete screen appears

## Progress
- [ ] Completed lesson count increases
- [ ] Accuracy changes after quiz
- [ ] Streak changes after practice

## Weak formulas
- [ ] Answer a question wrong
- [ ] Finish lesson
- [ ] Formula appears in Weak Formulas
- [ ] Practice weak formula again

## Class switching
- [ ] Change class from home
- [ ] Select Class 9
- [ ] Class 9 formulas appear

## Offline check
- [ ] Turn off Wi-Fi and mobile data
- [ ] App still works

## No internet permission check
- [ ] AndroidManifest.xml has no INTERNET permission

## Phone testing
- [ ] Test on Samsung phone
- [ ] Test on one older Android phone if possible
- [ ] Text is readable
- [ ] Buttons are easy to tap
